# pycemrg-carp
A high-level framework for cardiac model creation and simulation setup. 
Includes wrappers for the CARPentry/openCARP command-line tools.

> **Disclaimer:** This software is not affiliated with, sponsored by, or endorsed by the developers of CARPentry or openCARP. It is an independent, third-party library developed to facilitate the use of CARPentry command-line tools.
